# EXACT MODIFICATION GUIDE FOR ALL PAGES

## SUMMARY OF CHANGES NEEDED

You now have two new files in your Codeverse Project folder:
1. **shared.css** - Contains all animations, dark mode, hover effects
2. **shared.js** - Contains all functionality for dark mode, modals, animations

---

## FOR EVERY PAGE (home, about, blog1, blog2, contact, menu, portfolio)

### STEP 1: In the `<head>` section, add these two lines BEFORE closing `</head>`:

```html
<link rel="stylesheet" href="../shared.css">
```

### STEP 2: In the `<body>` section, add this button BEFORE closing `</body>`:

```html
<button class="theme-toggle">ðŸŒ™</button>
<script src="../shared.js"></script>
```

**IMPORTANT:** Make sure `<script src="../shared.js"></script>` is at the END, just before `</body>`

## IMPORTANT NOTES

1. **Path matters:** 
   - For pages in main folders (home, about, etc.), use `../shared.css`
   - For Navigation Page (in subfolder), use `../../shared.css`

2. **Script placement:** Always put `<script src="../shared.js"></script>` at the END before `</body>`

3. **Dark mode doesn't cover images** - Images stay visible in both modes

4. **Reservation button** - Works on all pages once you add the files and script

5. **Hamburger menu** - Already links to Navigation Page, no extra code needed

## IF YOU WANT TO CUSTOMIZE

**Change animation speed:** Edit `shared.css`, find `animation: slideUp 0.8s ease` and change `0.8s`

**Change dark mode colors:** Edit `shared.css`, find `:root` and change colors

**Change toggle button position:** Edit `shared.css`, find `.theme-toggle` and adjust `bottom` and `right` values
